Política de Privacidad

Aplicación Morevian Lawana

Fecha de entrada en vigor:

Bienvenido/a a Morevian Lawana (la "Aplicación"). Tu privacidad es importante para nosotros. Esta Política de Privacidad explica qué información (si la hay) recopilamos y cómo la usamos.

1. Información que NO Recopilamos

Morevian Lawana está diseñada pensando en tu privacidad. No recopilamos, almacenamos ni compartimos ninguna información personal identificable sobre ti. Esto incluye, pero no se limita a:

  • Nombre, dirección de correo electrónico, número de teléfono.
  • Datos de ubicación precisos o aproximados.
  • Información del dispositivo (más allá de lo necesario para las notificaciones push, gestionado por el sistema operativo y Firebase).
  • Datos de uso de la aplicación (qué himnos ves, cuánto tiempo usas la app, etc.).
  • Contactos, fotos u otros datos personales almacenados en tu dispositivo.

La aplicación funciona de manera local en tu dispositivo, mostrando el contenido del himnario moravo en miskito sin necesidad de conectarse a nuestros servidores para rastrear tu actividad.

2. Permiso de Notificaciones

El único permiso que Morevian Lawana puede solicitar es el permiso para enviar Notificaciones Push.

¿Por qué solicitamos este permiso?

Utilizamos este permiso exclusivamente para enviarte notificaciones relevantes a través del servicio Firebase Cloud Messaging (FCM), una plataforma de Google. Estas notificaciones pueden incluir:

  • Anuncios sobre nuevos himnos añadidos.
  • Actualizaciones importantes de la aplicación.
  • Recordatorios o mensajes relacionados con eventos o fechas significativas de la comunidad morava (si aplica).
  • Información sobre correcciones o mejoras.

Importante:

  • La concesión de este permiso es totalmente opcional. Puedes seguir utilizando todas las funciones principales de la aplicación (acceder al himnario) sin habilitar las notificaciones.
  • El uso de Firebase Cloud Messaging implica que tu dispositivo se registra en el servicio de Google para poder recibir dichas notificaciones. Google tiene su propia política de privacidad sobre cómo manejan los datos necesarios para este servicio (generalmente un token de registro anónimo del dispositivo). Nosotros, como desarrolladores de Morevian Lawana, no tenemos acceso a información personal tuya a través de este servicio, solo la capacidad de enviar mensajes a los dispositivos que han optado por recibirlos.
  • Puedes gestionar o revocar este permiso en cualquier momento a través de la configuración de notificaciones de tu dispositivo móvil (Android o iOS).

3. Servicios de Terceros

Como se mencionó, utilizamos Firebase Cloud Messaging (FCM), un servicio de Google, para la funcionalidad de notificaciones push. Te recomendamos revisar la política de privacidad de Firebase/Google para entender cómo manejan la información necesaria para este servicio.

Aparte de FCM para notificaciones (si las habilitas), la aplicación no integra otros servicios de terceros que recopilen información.

4. Seguridad

Aunque no recopilamos datos personales, tomamos medidas razonables para asegurar la integridad de la aplicación misma.

5. Cambios a esta Política de Privacidad

Podemos actualizar nuestra Política de Privacidad ocasionalmente. Te notificaremos cualquier cambio publicando la nueva Política de Privacidad en esta página dentro de la aplicación o mediante una notificación (si las tienes habilitadas). Se te aconseja revisar esta Política de Privacidad periódicamente para cualquier cambio.
